# Description

# Important:

After adding data to the appropriate tab, you must update the file name by adding a letter that is coresponding to the relevant reception (you can also add date to distinguish files- algoritm will ignore it). To minimalize errors when entering company names, there is a drop-down list that contains every company in the given reception.
